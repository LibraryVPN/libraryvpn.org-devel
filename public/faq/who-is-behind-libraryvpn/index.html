Who is behind LibraryVPN?

LibraryVPN is an IMLS funded project. It is being developed by a partnership of libraries including the Lebanon Public Libraries, the Westchester Library System, and the Cherry hill Public Library working with LEAP. This software is free and opensource, so anyone who wishes to can install it and run it free of charge. We are currently investigated the possibility of a hosted version of this software for libraries that wish to use it, but don’t want to host it for themselves.
